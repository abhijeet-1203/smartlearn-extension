# 🧠 SmartLearn AI - Chrome Extension

SmartLearn AI is an intelligent Chrome extension designed to enhance learning from online content, especially YouTube. It tracks user interactions, analyzes learning patterns, and provides AI-powered revision tools like quizzes and insights.

---

## 🚀 Features

- 📊 Track learning activity on YouTube
- 🧠 AI-powered learning assistance
- 📝 Generate quizzes for revision
- 📈 Visual insights using charts
- 🔔 Notifications & reminders
- 🎯 Personalized learning experience
